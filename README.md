# Kenna Real Estate — Lead Ops Dashboard

Analytics dashboard for Kenna Real Estate Group, powered by:
- Google Analytics 4 (GA4)
- Google Search Console (GSC)
- Follow Up Boss (FUB) CRM
- DataForSEO
- RelationalSEO

## Structure
- `index.html` — Dashboard HTML (import this into WP plugin)
- `kenna-dashboard.css` — All styles (referenced by HTML)
- `kenna-dashboard.js` — Data and chart renderers (referenced by HTML)

## WordPress Integration
Use the GitHub-to-WP plugin to import `index.html`. The CSS and JS
are loaded as external files from GitHub Pages automatically.

## Updating
1. Edit files locally or via AI
2. Push to GitHub
3. Press "Sync" in the WP plugin
4. Dashboard updates on your WordPress site
